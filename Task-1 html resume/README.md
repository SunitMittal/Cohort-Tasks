# Single Page Resume via HTML
A simple, clean HTML resume showcasing my professional experience, skills, projects, achievements, and education.

## Prerequisites
- Modern web browser (Chrome, Firefox, Edge, Safari)
- VS Code with Live Server extension

## Usage
- Open `resume.html` in any web browser to view the resume.
- Edit `resume.html` to update name, Contact, sections (About, Skills, Experience, Projects, Education).
- Save and refresh browser to preview changes.

## Features
- Semantic HTML5 structure
- Responsive design with inline styles
- Sections: Skills, Experience, Projects, Achievements, Education
- External links to GitHub, LinkedIn, and coding platforms
